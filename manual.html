MANUAL DE USUARIO

En este apartado vamos a ver cómo se utiliza el coche que hemos hecho.

En este manual vamos a explicar los pasos a partir de una vez que ya tenemos montado el coche, tanto la estructura como el cableado de los motores y sensores.

Básicamente lo que vamos a ver en este manual es cómo cargar el código desde el programa en el coche.

Paso 1

El primer paso que vamos a hacer es conectar el coche al ordenador. Para ello cogemos el cable USB de tipo C y lo conectamos a la ranura para este tipo de puertos de la placa ATMEGA328 y a un puerto USB del ordenador.

Cuando el coche esté conecado deberían iluminarse los componentes del coche, indicándonos que se ha conectado correctamente.

Paso 2

Ahora lo que vamos a hacer es abrir el programa Arduino IDE.

El logo del programa es este:

Interfaz del programa

Esta es la interfaz del programa Arduino IDE. Ahora vamos a ir por partes para ver para qué sirve cada cosa, cómo se sube el código al coche y todo lo que hay que hacer para ello.

Paso 3: Área de texto para introducir el código

El área blanca es donde tendremos que meter el código. Aquí declararemos los pines que número tienen, indicaremos si los sensores y motores son INPUTS o OUTPUTS y el loop, que es el código que se repetirá en función del código que hemos introducido.

Paso 4: Verificar el código

Este es uno de los pasos más importantes, ya que nos dirá si hay algún error en nuestro código y si lo hay nos indicará la línea dónde está. Los errores más frecuentes son dejarnos algún punto y coma ';' al final de una línea, un corchete mal cerrado o alguna variables mal declarada.

El botón de verificar es el primer botón de la barra de arriba.

Paso 5: Subir el código

En este paso lo que vamos a hacer es poner el código que hemos programado en el programa Arduino IDE al coche.

Subir el código es muy sencillo, únicamente hay que darle al botón subir, que es el segundo de la barra de arriba.

Paso 6: Encender el coche

Finalmente, una vez ya hemos subido el código al coche únicamente nos quedará encender el coche.

Antes de encender el coche hay que desconectar el cable USB tipo C de la placa ATMEGA328.

Para encender el coche bastará con poner la batería del coche en modo ON.
